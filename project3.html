Проект "Программирование на языке JavaScript"

Описание проекта
  • Создать простой калькулятор с использованием HTML, CSS и JavaScript.
  • Калькулятор должен быть оформлен с помощью CSS, чтобы сделать его визуально привлекательным.
  • Калькулятор должен быть реализован с использованием чистого и читабельного JavaScript-кода
  • Калькулятор должен быть протестирован, чтобы убедиться, что он работает так, как ожидается.
Использованные технологии и инструменты
  • HTML
  • CSS
  • JS
  • VScode
