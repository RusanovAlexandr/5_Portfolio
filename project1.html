Проект "Верстка сайта с применением CSS для стилизации HTML-элементов в соответствии с дизайном Figma"

Описание проекта
  • Создайте HTML-файл и CSS-файл в удобном для вас редакторе кода.
  • Используя дизайн Figma в качестве макета, создайте HTML-структуру сайта. Используйте семантические теги HTML, чтобы структурировать содержимое и сделать его доступным.
  • Используйте CSS для стилизации HTML-элементов в соответствии с дизайном Figma. Использовать селекторы CSS для выделения определенных элементов и применения стилей.
  • Протестируйте сайт в различных браузерах и на различных устройствах, чтобы убедиться, что он работает так, как ожидается.
Использованные технологии и инструменты
  • Figma
  • HTML
  • CSS
  • VSCode
